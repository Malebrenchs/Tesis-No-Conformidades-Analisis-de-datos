# QIPO-037: Técnicas e Instrumentos de Procesamiento de Datos

## Q
¿Cuáles son las técnicas e instrumentos utilizados para procesar los datos, transformar el registro original y preparar la base documental para el análisis?

## I
- Variable Independiente: Data Management  
  - Dimensión 1: Calidad de datos  
  - Dimensión 2: Modelado y normalización  
  - Dimensión 3: Analítica de datos  
- Proceso metodológico del estudio  
- Herramientas de transformación y visualización empleadas  

## P
Se identifican las herramientas necesarias para transformar la base documental original en un registro estructurado, normalizado y analíticamente útil.  
Se considera el uso de técnicas de Data Management para reorganizar la información, eliminar redundancias, establecer claves y permitir consultas analíticas.  
Se selecciona un instrumento especializado en visualización y análisis de datos para estructurar el pre–post.

## O
### **Técnicas**
- **Transformación documental**  
  Incluye:  
  - limpieza de datos,  
  - reorganización estructural,  
  - normalización relacional,  
  - separación de tablas (NC y eventos),  
  - estandarización de dominios,  
  - establecimiento de claves y trazabilidad.  

### **Instrumentos**
- **Power BI**  
  Utilizado para:  
  - estructurar el dataset,  
  - construir métricas,  
  - generar KPIs,  
  - crear visualizaciones pre–post,  
  - evaluar la capacidad analítica resultante.
